# UIDAI Aadhaar Data Analysis

This project presents a data-driven analysis of Aadhaar enrolment and demographic update patterns in India using anonymised datasets provided by UIDAI.

## Features
- Data cleaning and merging using Pandas
- State-wise and district-wise analysis
- Age-group distribution analysis
- Enrolment vs Update workload comparison
- Visualizations using Matplotlib

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Files
- analysis.ipynb -> Complete analysis code
- UIDAI_Aadhaar_Analysis_Shyamal_Jana.pdf -> Final report
